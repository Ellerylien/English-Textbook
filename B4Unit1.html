1-1形容詞原級基本句型

說明

不做任何變化，以原始樣貌呈現的形容詞，稱為形容詞的「原級」。

1be 動詞後接形容詞原級
主詞 + be 動詞 + 形容詞原級.
例 This amusement park is beautiful. 這個主題樂園很漂亮。
2形容詞原級後接名詞
主詞 + be 動詞 + a / an + 形容詞原級 + 名詞.
例 Window on China is a beautiful amusement park. 小人國是個很漂亮的主題樂園。

1-2「very + 形容詞原級」

說明

1. very 是副詞，用來修飾形容詞，表示強調。very 修飾形容詞原級，位於形容詞前面。

2. very、too、quite、pretty 等副詞用來修飾形容詞原級。其句型為：

A + be 動詞 + very / too / quite / pretty + 形容詞原級.
例 This amusement park is very beautiful. 這主題樂園非常漂亮。（very 修飾原級 beautiful）

1-3「as + 形容詞原級 + as」

說明

1. 兩者做比較時，如果比較的結果是相等的，可以用「as + 形容詞原級 + as」的句子表達。

2. 和 as、than 一樣，後面接人稱代名詞時要用主格，但口語中也常用受格。

1肯定句型
A + be 動詞 + as + 形容詞原級 + as +（B + be 動詞 / 受格）.
例 I am as old as she (is). = I am as old as her. 我跟她一樣大。
2否定句型
A + be 動詞 + not + as / so + 形容詞原級 + as + B（+ be 動詞）/ 受格.
例 He is not as rich as she (is). = He is not as rich as her. 他不像她一樣富有。

1-4形容詞比較級的規則與不規則變化

說明

形容詞分為三級：形容詞原級、形容詞比較級、形容詞最高級。形容詞的「比較級」用來表達兩者的比較，其中一方的程度高於另一方，稱為「優勢比較」。

1單音節或雙音節形容詞
單音節或雙音節形容詞比較級的規則變化
直接加 -er字尾為 e，直接加 -r「短母音 + 子音」
重複字尾加 -er		字尾為 y，去 y 加 -ier
cold → colder
dark → darker
weak → weaker		late → later
cute → cuter
nice → nicer		fat → fatter
big → bigger
thin → thinner		happy → happier
easy → easier
angry → angrier
2大多數雙音節和所有三個音節以上的形容詞，在原級前加 more
雙音節或多音節形容詞比較級的規則變化
beautiful → more beautiful
popular → more popular		famous → more famous
terrible → more terrible

多數兩個音節的形容詞，加 more 形成比較級，但也有些是加 -er，特別是字尾為 y 的形容詞，例如：lucky 的比較級為 luckier。

3有的形容詞比較級是不規則變化
形容詞比較級的不規則變化
good → betterbad → worselittle → lessmany / much → more

1-5形容詞比較級 than 的用法

說明

1. 形容詞比較級的句型使用 than 時，than 在句型中是連接詞，所以後面接人稱代名詞時，要使用主格加動詞；但在口語中有時會省略動詞，有時也可以使用受格。

2. 比較級的基本句型：

A + be 動詞 + 形容詞比較級 + than + B（+ be 動詞）.
例 Susan is taller than her mother (is). Susan 比她媽媽高。
例 My sister is taller than I (am). = My sister is taller than me. 我妹妹比我高。

3. 若表「較不⋯⋯」，形容詞前加 less 形成比較級。

例 The movie is less interesting than that one. 這部電影沒那部有趣。

1-6比較級用於同類型事物的比較

說明

同類的人或物才可以相比，不同類的不可以相比。

例 Lily’s math is better than Mandy.（×）
例 Lily’s math is better than Mandy’s (math).（○）
Lily 的數學比 Mandy 好。
💡

Lily’s math 不可以和 Mandy 這個人比較，應該和同類的 Mandy’s math 比。

1-7形容詞比較級「Which is...」

說明

1. 形容詞比較級的問句可以用 which 起首，which 可當「疑問代名詞」或「疑問形容詞」。

（1）當「疑問代名詞」時，which 後面不必加名詞，可直接加動詞。

例 Which is better, the red bag or the blue bag? 哪一個比較好，紅色包還是藍色包？

（2）當「疑問形容詞」時，which 後面加名詞，再加動詞。

例 Which bag is better, the red one or the blue one?
哪個包包比較好，紅色的還是藍色的？

2. 比較級的問句：

Which + 名詞
Which		+ is + 形容詞比較級,A or B?
例 A: Which animal is bigger, an elephant or a monkey? 哪種動物比較大，大象還是猴子？
B: An elephant is bigger than a monkey. 大象比猴子大。
例 A: Which is more interesting, math or science? 哪個比較有趣，數學還是科學？
B: Science is more interesting than math. 科學比數學有趣。

補充 one 的用法：one 單獨使用時，表示「不特定之物」，等於「a + 單數名詞」。one 之前有修飾語時，表「不特定之物」或「特定之物」，複數時要用 ones。

單數複數
a / an / the + 形容詞 / 序數onethe + 形容詞ones
this / thatthese / those

1-8形容詞比較級「Which has...」

說明

1. which 當「疑問代名詞」的句型：

Which + has + more + 名詞, A or B?
例 Which has more legs, the frog or the spider? 哪一個腳比較多，青蛙還是蜘蛛？

2. 回答 Which has... 的句型：

A + has / have + more / less + 名詞 + than + B.
例 The spider has more legs than the frog. 蜘蛛的腳比青蛙多。
例 The frog has less legs than the spider. 青蛙的腳比蜘蛛少。

1-9修飾形容詞比較級的副詞

說明

1. 修飾形容詞比較級的副詞：修飾形容詞比較級時，需要用 even、much、a little、a lot、lots 等副詞來修飾。

2. 修飾形容詞原級的副詞與修飾形容詞比較級的副詞不同，如下：

修飾形容詞原級的副詞very, too, quite, pretty
修飾形容詞比較級的副詞even, much, a little, a lot,（lots 不正式）
例 The building is very tall. 這棟建築物很高。
例 Samuel is much taller than his brother. Samuel 比他弟弟高得多。
例 The skirt is a little cheaper than the jeans. 這裙子比牛仔褲便宜一點。

進階練習

1-10「The + 形容詞比較級, the + 形容詞比較級」

說明

1.「The 比較級 ..., the 比較級 ...」是表示「越⋯⋯，就越⋯⋯」。可和形容詞或副詞連用，表示「原因和結果」，也就是「當一方變化時，另一方也隨之改變」。

2.「The 比較級 ..., the 比較級 ...」的句型：

The 比較級 + A + 動詞 ..., the 比較級 + B + 動詞.
例 The harder you study, the better grades you will get. 你越用功，就越會拿到更好的成績。

1-11「比較級 + and + 比較級」

說明

「比較級 + and + 比較級」的句型如下：

主詞 + be 動詞 / get / become + 比較級 + and + 比較級.
例 The weather is colder and colder. 天氣越來越冷了。
例 Mandy is more and more beautiful. Mandy 越來越美了。
